การเขียนติดต่อใช้งานกับ FTP Server

ภาษาไพธอน

เซิร์ฟเวอร์ FTP เป็นแอปพลิเคชันซอฟต์แวร์ที่เปิดการทำงานการโอนย้ายไฟล์จากอุปกรณ์เครื่องหนึ่ง (เช่น คอมพิวเตอร์ที่ใช้ระบบปฏิบัติการ Mac, Windows หรือ Linux) ไปยังอีกเครื่องหนึ่ง

  • โมดูล pyftpdlib เป็นโมดูลสำหรับใช้สร้าง FTP server บนภาษา Python เข้ากันได้กับมาตรฐาน RFC-959.

มีขนาดเล็ก เร็ว มีประสิทธิภาพไม่ต่างกันกับ proftpd สนับสนุน FTPS (RFC-4217), IPv6 (RFC-2428), Unicode file names (RFC-2640),MLSD/MLST commands (RFC-3659). รองรับทั้ง Python 2 และ Python 3 ใช้งานได้ทั้ง UNIX และ Windows ใช้ MIT license
